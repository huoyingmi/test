我歼20与加油机进行空中加油合练 形成完全作战能力

8月12日，央视军事报道栏目中做了一个歼-20形成战斗力，并与歼-16和歼-10C联合进行空战战术探索的专题报道。在此专题中，央视也提到了专业蓝军锤炼一线部队的事宜，并提出歼-20未来或许有可能装备到空军的专业蓝军部队中。这也使得，此前传的沸沸扬扬的“歼-20已经列装到中国空军蓝军部队”的谣言不攻自破。在此谣言之下，还衍生出了很多关于“演习实况”的段子。其中荒诞离谱者有之；有鼻子有眼，让人看了如身临其境者亦有之。本期《出鞘》我们就来结合“蓝军歼-20”的四条段子及国外空军对抗演习来探讨一下，未来歼-20在中国空军的演习和体系对抗中将要发挥的作用。（查看完整内容搜索微信公众号：sinamilnews）

近日网上出现歼-20和伊尔-78空中加油机合练的图片，表明歼-20可能已经开始进行空中加油训练，飞机已经形成完全作战能力，正在探索如何执行远程打击任务，这是中国空军又一个历史性跨越。

我们知道空中加油是一个风险比较大课目，对于飞机远程作战至关重要，在新机形成战斗力众多课目之中比较靠后，因此歼-20进行空中加油课目训练也标志着飞机已经形成完全作战能力，开始向多用途任务扩展。
